Comandos Basicos de Git GIT INIT: Inicializa el repositorio GIT ADD: AÃ±adir los cambios a la zona temporal GIT COMMIT -M "": Confirmar los cambios para la creacion de una version GIT STATUS: Nos muestra el estado en el que se encuentra el repositorio GIT LOG: El historial completo de nuestras versiones GIT LOG --ONELINE: El historial resumido de nuestras versiones GIT DIFF VA VB: Diferencia entre una version con otra GIT CHECKOUT: Nos permite realizar un retroceso en el tiempo a una version anterior GIT RESET: Nos permite regresar en el tiempo pero manera mas peligrosa GIT RESET --SOFT: Nos permite regresar en el tiempo y nos permite modificar nuestro pasado para que se fusione con el futuro (todos los cambios posteriores a la version regresada se mantendra en la zona temporal) GIT RESET --HARD: Nos permite regresar en el tiempo pero al regresar al pasado ese pasado se convierte en nuestro presente (Perdemos todas las versiones posteriores a la version regresada) GIT BRANCH nombreRama: Nos permite crear una ramificacion del codigo principal en donde podemos tener diversos cambios en el codigo que no afectaran a la version Principal (main, master) GIT BRANCH: Nos mostrar la lista de ramificaciones que tenemos
