Comandos de Git

¿Qué es Git?

Git es un sistema de control de versiones que se utiliza para el seguimiento de cambios en archivos y proyectos de software. Es especialmente útil para colaborar en proyectos con otras personas.

Comandos básicos de Git

git init

Este comando se utiliza para inicializar un nuevo repositorio de Git en un proyecto.

git add

Este comando se utiliza para agregar archivos o cambios específicos a un commit.

git commit

Este comando se utiliza para guardar los cambios realizados en un archivo o conjunto de archivos.
